# Smart Queue System

A modern, presentation-ready queue management solution for clinics, reception desks, and public service environments.

This project is built as a polished React/Vite interface for managing patient flow, queue status, public display views, and operational reporting in a clear, professional way.

## Why it stands out

- Fast and intuitive staff workflow
- Clear patient queue overview and status handling
- Simple call-next-patient flow
- Public display view for waiting areas
- QR-based status page for patients
- Reporting and export-ready insights

## Core features

- Demo authentication and role-based views
- Patient check-in and queue management
- Search, filtering, and status updates
- Room assignment and priority handling
- Public display and patient status sharing
- Reports and CSV-style export options
- Settings for departments, rooms, and queue states

## Tech stack

- React + TypeScript
- Vite
- React Router
- Lucide icons
- Vitest, ESLint, and Playwright verification

## Run locally
